# aws-cli-go
## ms-clear

Utility to clear MediaStore container folder


```bash
Usage:
  ms-clear [OPTIONS]

Application Options:
  -p, --profile=  AWS Profile
  -r, --region=   AWS Region (default: us-east-1)
  -e, --endpoint= MediaStore Endpoint
  -f, --folder=   MediaStore folder
      --force     Force operation without prompting.

Help Options:
  -h, --help      Show this help message
```
